# InputNumber

```typescript
customElements.define(`sc-input-number`, InputNumber)
```
```html
<sc-input-number value="5" />
```

## Attributes

| name       |  type   | is reactive | description |
| :--------: | :-----: | :---------: | :---------: |
| `value`    | number  |     yes     | the value   |

## Events

| name     | description         |
| :------: | :------------------ |
| `input`  | dispatched on input |
